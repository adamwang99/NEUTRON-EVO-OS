# Learned Skills

This directory accumulates **learned skills** — insights, patterns, and techniques discovered during sessions that weren't part of any predefined skill.

## How it works

When a session surfaces a reusable pattern or approach that doesn't fit an existing skill category, it's captured here as a **learned note**.

## Format

```markdown
# Learned: [Short description]

## Pattern
What was discovered.

## When to apply
Context for reuse.

## Example
Brief example if applicable.
```

## Maintenance

Review this directory monthly. Entries that become stable and recurring should be promoted into the appropriate `skills/core/` category.
